# ComputingLogic
For Eclipse installation on linux system :
1) Download Eclipse from http://eclipseclp.org/Distribution/7.0_33/
2) UnZip it and run script RUNME
3) Add enviroment variable to PATH

For Picat installation on linux system :
1) Download Eclipse from http://picat-lang.org/download.html
2) UnZip it and run script picat


All the files without extension are .txt files and contain sample input and output for the programs. 
for e.g. alien corresponds to alien.ecl
